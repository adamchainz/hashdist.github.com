HashDist

Build it Once

  • build complete software stacks
    {% highlight bash %} $ hit build scipy-stack.yaml {% endhighlight %}
  • across operating systems
  • version them
    {% highlight bash %} $ git checkout scipy-stack.yaml {% endhighlight %}
  • use packages from the system
    {% highlight yaml %} python: use: host-python {% endhighlight %}
  • or build your own
    {% highlight yaml %} python: build_with: | bzip2, openssl, sqlite {% endhighlight %}
  • and customize them
    {% highlight yaml %} petsc: build_with: | parmetis download: | superlu, scalapack, mumps coptflags: -O2 link: shared debug: false {% endhighlight %}
  • deploy production code
    {% highlight yaml %} extends: [distutils_package] dependencies: run: [pyzmq, tornado] sources: - url: http://archive.ipython.org/release/2.0.0/ipython-2.0.0.tar.gz key: tar.gz:j2b4bd4nn7iorbemnd6a5mxzwbkptn2c {% endhighlight %}
  • or your own customizations
    {% highlight yaml %} extends: [distutils_package] dependencies: run: [pyzmq, tornado] sources: - url: https://github.com/ahmadia/ipython.git key: git:68f6519cdfd23581ed7358fa1941ae1bbd76e5b3 {% endhighlight %}

Get Started

{% highlight bash %} $ pip install hashdist $ git clone https://github.com/hashdist/hashstack.git $ ./hashdist/bin/hit build hashstack/examples/proteus.linux2.yaml {% endhighlight %}
